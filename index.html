HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element defines the title of the webpage.
  • The style element defines the CSS styles associated with a webpage.
  • The p element is used to wrap paragraphs and blocks of text while ul, ol, and li elements are for unordered lists.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The CSS box model refers to the order of alignment and coloring, from outermost to innermost: margin, border, padding, content.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git checkout main: switch to the main branch.
  • git pull origin main: synchronize local main with remote main.
  • touch file-name.type: creates a new file in current directory.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A conditional statement executes a block of code if the condition is truthy; if the condition is falsy, it does not execute.
  • A for Loop has a starting point, a condition, and a modifier such as x++; it is used to cover a list or collection of elements.
  • A function is a block of code with a specific name and must be called to execute its contents.
